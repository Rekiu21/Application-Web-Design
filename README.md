## ¿Qué es Markdown?
Markdown es un lenguaje de marcado ligero ("lightweight markup language") que permite dar formato a texto plano de forma rápida y legible. Se usa mucho para:

- Documentar proyectos (por ejemplo, en archivos `README.md`)
- Escribir guías y reportes en plataformas como GitHub
- Crear listas, encabezados, enlaces, código y tablas sin necesidad de editores complejos

En resumen: Markdown sirve para escribir documentación clara y bien estructurada usando solo texto.

## Mini tutorial de Markdown (ejemplos)

### Encabezados
```md
# Encabezado 1
## Encabezado 2
### Encabezado 3
```

### Negritas y cursivas
```md

*cursivas*
```

### Listas
```md
- Elemento A
- Elemento B
  - Sub-elemento
```

### Enlaces
```md

```

